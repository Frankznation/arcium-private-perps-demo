# Arcium Private Perps - Interactive Demo

Standalone Next.js demo application for Arcium Private Perps platform.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

### Vercel
```bash
vercel
```

### Netlify
Connect GitHub repository and deploy.

## Features

- ✅ Interactive position opening
- ✅ Privacy encryption simulation
- ✅ PnL calculator
- ✅ Liquidation risk checker
- ✅ No backend needed
- ✅ No environment variables needed
